Chart Recorders / Data Loggers

VIG offers chart recorders and data loggers to meet customer-specific recording and acquisition requirements, including single-pen, multi-pen, and digital data acquisition options.

DataChart 2000 Paperless Data / Chart Recorder

The DataChart 2000 supports 2, 4, 6, or 12 direct universal isolated inputs selectable for DC voltage, DC current, and thermocouples. Touch-screen control supports fast setup and operation.

Its 5.6 inch active matrix TFT color display supports clear viewing, with a blue monochrome display available as an economical option. Data can be stored in nonvolatile internal RAM and downloaded automatically or on demand to removable media.

Communication options include RS232C with modem support, RS485 MODBUS for network integration, and 10BaseT Ethernet for LAN or WAN access using TCP/IP.

DataTaker 4000 Data Logger

The DataTaker 4000 series includes compact, self-powered data loggers for 1, 2, 4, and 5 channel analog voltage and current loop signals.

  • 12-bit resolution for accurate and sensitive readings.
  • Nonvolatile memory for data protection.
  • Automatic conversion to engineering units.
  • Battery-powered operation for long-term installation needs.

Service note: VIG can help match recording hardware to required channels, signal types, data access, and reporting needs.
